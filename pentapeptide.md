# Pentapeptide

| Property | Value |
|:---------|------:|
| Code | Gromacs |
| Integrator | Langevin |
| Integrator time step | 2 fs |
| Simulation time | 500 ns |
| Frame spacing | 100 ps |
| Temperature | 300 K |
| Solvation | implicit (GBSA) |
| Electrostatics | cutoff (1.5 nm) |
| Constraints | all bonds between hydrogens and heavy atoms |
